# # 📊 지역별 연령별 인구 구조 대시보드

행정안전부 **주민등록 연령별 인구현황(월간)** 데이터를 기반으로, 지역별 인구 구조를 탐색하고 분석하는 Streamlit 웹앱입니다.

## 🚀 데모

Streamlit Community Cloud에 배포해서 바로 사용할 수 있습니다.

## 📁 파일 구성

```
.
├── app.py                                          # 메인 앱 코드
├── requirements.txt                                # 필요 패키지 목록
└── 202606_202606_연령별인구현황_월간__3_.csv        # 원본 데이터 (app.py와 같은 폴더)
```

> ⚠️ 데이터 파일명은 `app.py` 안의 `DATA_FILE` 상수에 고정되어 있습니다. 매월 데이터를 갱신할 경우 새 CSV 파일명으로 이 상수를 수정하거나, 파일명을 동일하게 맞춰서 교체해주세요.

## ✨ 주요 기능

### 1. 지역별 연령 구조 꺾은선 그래프
- 사이드바에서 **검색창 + 선택 드롭다운**으로 원하는 지역(시/도, 시/군/구, 읍/면/동 모두 지원)을 찾을 수 있습니다.
- 기준 지역 외에 여러 지역을 추가로 선택해 **동시 비교**가 가능합니다.
- 계/남/여 성별을 선택적으로 표시할 수 있습니다 (실선=계, 점선=남, 파선=여).
- Plotly 기반으로 0세 ~ 100세 이상까지 연령별 인구수를 인터랙티브하게 확인할 수 있습니다.

### 2. 전국 유사 지역 Top 5
- 선택한 지역과 **연령대별 인구 비중(%)의 모양**이 가장 비슷한 지역 5곳을 전국에서 찾아줍니다.
- 총인구 규모 차이를 배제하기 위해 인구수가 아닌 **비율(정규화된 분포)** 로 비교합니다.
- 비교 대상은 선택 지역과 **같은 행정 단위**(시/도 ↔ 시/도, 읍/면/동 ↔ 읍/면/동)로 제한해 의미 있는 비교가 되도록 했습니다.
- 유클리드 거리 기반으로 유사도를 계산하며, 그래프와 요약 테이블로 함께 제공됩니다.

### 3. 청년 인구(만 19~34세) 최다 동네 Top N
- 읍/면/동 단위에서 만 19~34세 인구를 집계해 **청년 인구가 가장 많은 동네**를 순위로 보여줍니다.
- **"청년 인구수"(절대량)** 또는 **"청년 인구 비율(%)"** 중 정렬 기준을 선택할 수 있습니다.
- Top N 개수(5~30개)와 시/도 필터를 자유롭게 조정할 수 있습니다.
- Plotly 가로 막대 그래프 + 요약 테이블로 결과를 확인할 수 있습니다.

## 🛠 기술 스택

- [Streamlit](https://streamlit.io/) — 웹앱 프레임워크
- [Plotly](https://plotly.com/python/) — 인터랙티브 그래프
- [Pandas](https://pandas.pydata.org/) / [NumPy](https://numpy.org/) — 데이터 처리 및 유사도 계산

## 💻 로컬 실행 방법

```bash
# 1. 저장소 클론
git clone <이 레포지토리 URL>
cd <레포지토리 폴더>

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 앱 실행
streamlit run app.py
```

## ☁️ Streamlit Community Cloud 배포 방법

1. GitHub에 `app.py`, `requirements.txt`, 데이터 CSV 파일을 함께 커밋합니다. (모두 같은 폴더/루트에 위치해야 합니다)
2. [share.streamlit.io](https://share.streamlit.io) 에서 New app 생성
3. 방금 만든 GitHub 레포지토리 선택
4. Main file path에 `app.py` 입력
5. Deploy 클릭
